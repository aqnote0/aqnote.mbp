A Linux driver for the Facetime HD camera on Apple MacBook Pro

I already detailled the installation of GNU/Linux Debian on an Apple MacBook Pro. Unsupported webcam was one of the most annoying part. yes I wrote “was”, because a driver is under active development.

Of course, I tested, and approved it.
But, even if manual installation of both driver and firmware works fine, I still prefer managing that kind of file with Debian packages.

That’s the reason why I startes the packaging stuff.
Though it’s still an incomplete and in progress work, you can find it on my github repository.

You will be able to build two Debian packages:

  • broadcom-pcie-webcam-dkms which provides the facetimehd driver source compatible with DKMS
  • firmware-broadcom-pcie-webcam which provides the binary firmware of the camera.
    It’s downloaded from Appel .dmg updates.

Building the package

Stating you already a working dev environment, it’s very easy to build the Debian packages:

# git clone https://github.com/jbfavre/bcwc_pcie.git
# cd bcwc_pcie
# git clone debian
# gbp buildpackage -us -uc --git-debian-branch=debian

Installation

Once you have built the package, you can install them:

$ dpkg -i ../firmware-broadcom-pcie-webcam_0.0.1*.deb ../broadcom-pcie-webcam-dkms_0.0.1*.deb

If everything goes fine, you should have following files:

# ls -l /lib/firmware/facetimehd/
total 1396
-rw-r--r-- 1 root root 1425412 mars   3 15:06 firmware.bin
# ls -l /lib/modules/4.4.0-1-amd64/updates/dkms/
total 7488
-rw-r--r-- 1 root root  165016 mars   3 15:07 facetimehd.ko

If you find the files, you can now load the module and test it:

$ modprobe facetimehd

Enjoy :)
